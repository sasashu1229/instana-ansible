# RPMファイル配置ディレクトリ

このディレクトリにPython RPMパッケージファイルを配置してください。

## 必要なファイル

- `python3.9-3.9.6-1.aix7.1.ppc.rpm`

## 配置方法

1. Python 3.9のRPMパッケージファイルをこのディレクトリにコピーしてください
2. ファイル名が `vars/python_vars.yaml` で指定されている名前と一致していることを確認してください

```bash
# 例: RPMファイルをこのディレクトリにコピー
cp /path/to/python3.9-3.9.6-1.aix7.1.ppc.rpm ./files/
```

## 注意事項

- RPMファイルは実行前に必ず配置してください
- ファイル名を変更する場合は、`vars/python_vars.yaml` の `python_rpm_file` 変数も更新してください
- このディレクトリには実際のRPMファイルのみを配置し、バージョン管理システムには含めないでください
